Use on the webの所から、コピーします。

head要素の中に先程コピーした物をペーストします。

            
                <head>
                    <link rel="preconnect" href="https://fonts.googleapis.com">
                    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
                    <link href="https://fonts.googleapis.com/css2?family=Zen+Kurenaido&display=swap" rel="stylesheet">
                </head>

次にCSS用の物もコピーします。

            
                .Zen-Kurenaido{ /*分かりやすくするためフォント名にするのが無難 */
                    font-family: 'Zen Kurenaido', sans-serif;
                }

変えるにはクラス名に先ほど登録した名前を書くだけでできる。

デザインをレスポンシブデザインにする

レスポンシブなデザインにすることでスマホ用とPC用で別々にhtmlファイルを用意しなくても大丈夫になる。

以下二つの方法があります。

1:ビューポート(viewport)を設定する

            
                <head>
                    <!-- ビューポートの設定 -->
                    <meta name="viewport" content="width=device-width,initial-scale=1">
                </head>

表示領域を、各端末の画面幅に合わせて、いい感じに調整してくれます。

2:メディアクエリを設定する

▼コード(CSS)

            
                /* 768~1020pxの幅に対応した処理(主にタブレットや画面の小さいPC用) */
                @media (max-width: 1020px) and (min-width: 768px) { 
                    /* ここに処理を書く */
                }
                /* スマホ用 */
                @media (max-width: 767px) { 
                    /* ここに処理を書く */
                }

▼コード(HTML)

            
                <div class="Two-column">
                    <div class="left">
                        <p>左側のコンテンツ</p>
                    </div>
                    <div class="right">
                        <p>右側のコンテンツ</p>
                    </div>
                </div>

このような、htmlを書いたとする。

▼コード(CSS)

            
                .Two-column {
                    display: flex;
                }
                .left {
                    width: 50%;
                    background-color: #bbfaca;
                }
                .right {
                    width: 50%;
                    background-color: #bbdffa;
                }

                @media screen and (max-width: 767px) {
                    .Two-column {
                        display: block;
                    }
                    .left {
                        width: 100%;
                    }
                    .right {
                        margin-top: -1rem;
                        width: 100%;
                    }
                }
