CSSでデザインを調整する

CSSを活用すると、HTML内にある関連箇所をいちいち直さなくても、一括して変更可能になります。

▼コード(HTML)

            
                <!doctype html>
                <html>
                    <head>
                        <title>ポートフォリオ</title>
                        <link href="CSSのパス" rel="stylesheet" type="text/css">
                    </head>
                    <body>
                        <div class="blue">
                            Hello, World
                        </div>
                    </body>
                </html>

▼コード(CSS)

            
                .blue{
                    background-color: blue;
                }

▼htmlの見た目

Hello, World

見た目は、ここと同じになる。

IDは、そのページに一つしかないものにつけます。CLASSは何個でもOKです。

もっと沢山のフォントを見てみる

Google Fontというのを利用して、更にフォントを見てみる。

https://fonts.google.com/にアクセスして下さい。

好きなフォントを選択してください。

選択すると上記画像のようになるので、セレクトの部分をクリック

Use on the webの所から、コピーします。

head要素の中に先程コピーした物をペーストします。

            
                <head>
                    <link rel="preconnect" href="https://fonts.googleapis.com">
                    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
                    <link href="https://fonts.googleapis.com/css2?family=Zen+Kurenaido&display=swap" rel="stylesheet">
                </head>

次にCSS用の物もコピーします。

            
                .Zen-Kurenaido{ /*分かりやすくするためフォント名にするのが無難 */
                    font-family: 'Zen Kurenaido', sans-serif;
                }

変えるにはクラス名に先ほど登録した名前を書くだけでできる。
