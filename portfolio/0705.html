ページの一番上に戻るようにしてみる

今回は普段は見えていなくて、一定の距離スクロールしたら画像が表示され、クリックしたらページの一番上に戻るようにしてみる

▼コード(HTML)

            
                <div id="pagetop" hidden>
                    <img src="img/TOP.png" width="80">
                </div>
これをbodyタグの下の方に追加する。

▼コード(CSS)

            
                #pagetop {
                    position: fixed;
                    /* 位置を決める(自由でOK) */
                    right: 10px;
                    bottom: 0px;
                    /* カーソルをポインタに変更 */
                    cursor: pointer;
                    /* 常に手前に表示 */
                    z-index: 10;
                }

▼コード(JS)

            
                $(function(){
                    var topBtn=$('#pagetop');
                    topBtn.hide();
                   
                    //ボタンの表示設定
                    $(window).scroll(function(){
                      if($(this).scrollTop()>600){
                        //---- 画面を600pxスクロールしたら、ボタンを表示する
                        topBtn.fadeIn();
                      }else{
                        //---- 画面が600pxより上なら、ボタンを表示しない
                        topBtn.fadeOut();
                      } 
                    });
                   
                    //ボタンをクリックしたら、スクロールして上に戻る
                    topBtn.click(function(){
                        $('body,html').animate({
                        // スクロールのスピード
                        scrollTop: 0},500);
                        return false;
                    });
                  });

またこれだけでは、使えないので

            
                <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

という、jQueryのライブラリを読み込んでください。

▼コード(JS)

                    {
                        autoplay: true, //自動的に動き出すか。初期値はfalse。
                        infinite: true, //スライドをループさせるかどうか。初期値はtrue。
                        speed: 500, //スライドのスピード。初期値は300。
                        slidesToShow: 3, //スライドを画面に3枚見せる
                        slidesToScroll: 1, //1回のスクロールで1枚の写真を移動して見せる
                        prevArrow: '<div class="slick-prev"></div>', //矢印部分PreviewのHTMLを変更
                        nextArrow: '<div class="slick-next"></div>', //矢印部分NextのHTMLを変更
                        centerMode: true, //要素を中央ぞろえにする
                        variableWidth: true, //幅の違う画像の高さを揃えて表示
                        dots: true, //下部ドットナビゲーションの表示
                    }
                );

▼コード(JS)

<time datetime="2023-07-05"></time></p>が14日以内の時にnewというラベルをつける。 

            
                $(function () {
                    $('time').each(function(){
                      // 現在日時
                      var current = new Date();
                      // 14日前のミリ秒
                      var range_ms = current.getTime() - (14 * 24 * 60 * 60 * 1000);
                  
                      // 更新日時
                      var modified = new Date($(this).attr('datetime'));
                      // 更新日時のミリ秒
                      var modified_ms = modified.getTime();
                  
                      if (range_ms < modified_ms){
                        $(this).after('<li><strong class="label label-danger">New</strong></li>');
                      }
                    });
                });
