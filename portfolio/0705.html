ボタンを押したら入力した文字をアラートするようにする

▼コード(HTML)

            
                <div>
                    <input type="text" id="textBox1" value="テキストを入力してね">
                    <button onclick="clickTextBox1()">決定</button>
                </div>

▼htmlの見た目

このままでは何も起こらないのでスクリプトを書く。

▼コード(JS)

            
                // 関数名は、 <button onclick="clickTextBox1()">のonclickの部分
                    function clickTextBox1(){
                        // 探索するデータ
                        //  <input type="text" id="textBox1" value="テキストを入力してね">のid部分
                        getTextBox1 = document.getElementById("textBox1");
                        // 値を取得する
                        text1 = getTextBox1.value;
                        // text1の内容をアラーとする
                        alert(text1);
                    }

次は、htmlの内容を直接書き換えてみる。

                <div>
                    <input type="text" id="textBox2" value="テキストを入力してね">
                    <button onclick="clickTextBox2()">決定</button>
                </div>
                <div id="displayTextBox2">
                    変換テスト
                </div>

変換テスト

▼コード(JS)

            
                // 関数名は、 <button onclick="clickTextBox2()">のonclickの部分
                    function clickTextBox2(){
                        // 探索するデータ
                        //  <input type="text" id="textBox2" value="テキストを入力してね">のid部分
                        getTextBox2 = document.getElementById("textBox2");
                        // 値を取得する
                        text2 = getTextBox2.value;
                        text2 = "ほらね？「" + text2 + "」に変わってるでしょ？";
                    
                        DisplayText2 = document.getElementById("displayTextBox2");
                        // HTMLに流し込む
                        DisplayText2.innerHTML = text2;
                    }

ページの一番上に戻るようにしてみる

今回は普段は見えていなくて、一定の距離スクロールしたら画像が表示され、クリックしたらページの一番上に戻るようにしてみる

▼コード(HTML)

            
                <div id="pagetop" hidden>
                    <img src="img/TOP.png" width="80">
                </div>
これをbodyタグの下の方に追加する。

▼コード(CSS)

            
                #pagetop {
                    position: fixed;
                    /* 位置を決める(自由でOK) */
                    right: 10px;
                    bottom: 0px;
                    /* カーソルをポインタに変更 */
                    cursor: pointer;
                    /* 常に手前に表示 */
                    z-index: 10;
                }

▼コード(JS)

            
                $(function(){
                    var topBtn=$('#pagetop');
                    topBtn.hide();
                   
                    //ボタンの表示設定
                    $(window).scroll(function(){
                      if($(this).scrollTop()>600){
                        //---- 画面を600pxスクロールしたら、ボタンを表示する
                        topBtn.fadeIn();
                      }else{
                        //---- 画面が600pxより上なら、ボタンを表示しない
                        topBtn.fadeOut();
                      } 
                    });
                   
                    //ボタンをクリックしたら、スクロールして上に戻る
                    topBtn.click(function(){
                        $('body,html').animate({
                        // スクロールのスピード
                        scrollTop: 0},500);
                        return false;
                    });
                  });

またこれだけでは、使えないので

            
                <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

という、jQueryのライブラリを読み込んでください。

画像をスライドさせる

▼コード(HTML)

            
                <ul class="slider">
                    <li><img src="画像パス" alt=""></li>
                    <li><img src="画像パス" alt=""></li>
                    <li><img src="画像パス" alt=""></li>
                    <li><img src="画像パス" alt=""></li>
                </ul>

▼コード(CSS)

            
                .slider {
                    width: 94%;
                    margin: 0 auto;
                }
                  
                .slider img {
                    width: 60vw;
                    height: auto;
                }
                  
                .slider 
                .slick-slide {
                    transform: scale(0.8);
                    transition: all 0.5s;
                    opacity: 0.5;
                }
                  
                .slider
                .slick-slide.slick-center {
                    transform: scale(1);
                    opacity: 1;
                }
                  
                /*矢印の設定*/
                  
                /*戻る、次へ矢印の位置*/
                .slick-prev,
                .slick-next {
                    position: absolute;
                    top: 42%;
                    cursor: pointer;
                    outline: none;
                    border-top: 2px solid #666;
                    border-right: 2px solid #666;
                    height: 15px;
                    width: 15px;
                }
                  
                .slick-prev {
                    left: -1.5%;
                    transform: rotate(-135deg);
                }
                  
                .slick-next {
                    right: -1.5%;
                    transform: rotate(45deg);
                }
                  
                /*ドットナビゲーションの設定*/
                  
                .slick-dots {
                    text-align: center;
                    margin: 20px 0 0 0;
                }
                  
                .slick-dots li {
                    display: inline-block;
                    margin: 0 5px;
                }
                  
                .slick-dots button {
                    color: transparent;
                    outline: none;
                    width: 8px;
                    height: 16px;
                    display: block;
                    border-radius: 50%;
                    background: #ccc;
                }
                  
                .slick-dots
                .slick-active button {
                    background: #333;
                }

▼コード(JS)

            
                $('.slider').slick(
                    {
                        autoplay: true, //自動的に動き出すか。初期値はfalse。
                        infinite: true, //スライドをループさせるかどうか。初期値はtrue。
                        speed: 500, //スライドのスピード。初期値は300。
                        slidesToShow: 3, //スライドを画面に3枚見せる
                        slidesToScroll: 1, //1回のスクロールで1枚の写真を移動して見せる
                        prevArrow: '<div class="slick-prev"></div>', //矢印部分PreviewのHTMLを変更
                        nextArrow: '<div class="slick-next"></div>', //矢印部分NextのHTMLを変更
                        centerMode: true, //要素を中央ぞろえにする
                        variableWidth: true, //幅の違う画像の高さを揃えて表示
                        dots: true, //下部ドットナビゲーションの表示
                    }
                );

これも利用するためにいくつかスクリプトを追加する。

            
                // head要素に追加する。
